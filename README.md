# BalkanBiz TV

Open-source kuriran katalog poslovnih YouTube kanala iz balkanskih zemalja — Hrvatska, Srbija, BiH, Slovenija, Makedonija, Crna Gora.

Bez algoritma, bez praćenja, bez oglasa. Samo poslovni sadržaj na našem jeziku.

## Live

https://balkanbiz-tv.vercel.app (uskoro)

## Stack

- Next.js 16 (App Router) + React 19
- Tailwind CSS v4
- Static export — sve renderira u build time
- Hostan na Vercelu

## Doprinos

Predložite novi kanal kroz [GitHub issues](https://github.com/toma2502/balkanbiz-tv/issues/new) ili otvorite pull request s izmjenom `data.json`.

Kriteriji za uvrštavanje:
- Kanal je iz neke od 6 ciljanih zemalja
- Sadržaj je primarno poslovne, marketinške, financijske, tech ili leadership prirode
- Kanal je aktivan (objava u zadnjih 6 mjeseci)
- Sadržaj je na hrvatskom, srpskom, bosanskom, slovenskom, makedonskom ili crnogorskom jeziku
